Jupyter Book

- Jupyter Book is a system for creating reproducible, reusable and interactive documents - Source is written in MyST Markdown (an extension of common mark) or Jupyter Notebook - Typography features - Lists - Tables - Figures - Maths - Tabs, dropdowns - More … - Interactive content - Executable content in - Cells in a Jupyter Notebook (`.ipynb`) - `code-cell` blocks in Markdown files - Inline code using the `eval` role - Can execute at build time with a local Jupyter server and kernel - Can execute on the page by connecting to a Jupyter server and kernel (_e.g._ MyBinder) - Powerful and flexible plugin system - Reusable: content can be referenced or embedded from other Jupyter Books - Exports to HTML, PDF (LaTeX and Typst), word, JATS - Through intermediate AST - History: - Jupyter Book 1, based on Sphinx documentation engine, supported MyST Markdown input through myst-parser - Jupyter Book 2, a distribution of MyST MD document engine
