Jupyter Book

Create documents and knowledge bases that are reusable, reproducible, and interactive.
  • Features for beautiful typesetting, in many domains
  • Builds from MyST Markdown or Jupyter Notebooks
- Jupyter Book is a system for creating reproducible, reusable and interactive documents - Many features - Typography - Structure - Lists - Tables - Tabs, dropdowns - Technical - Maths - Code - Diagrams - Media - Figures - Videos - More ‚Ä¶ - Source is written in MyST Markdown (an extension of common mark) or Jupyter Notebook

Executable content

  • Jupyter Notebooks
  • code-cell blocks in Markdown
  • Inline statements using eval in Markdown

                                        # Title

                                        ```{code-cell} python
                                        hello = "hello"
                                        there = "there"
                                        phrase = f"{hello}, {there}!"
                                        print(phrase)
                                        ```
                                    

                                    # Title

                                    1 + 1 is {eval}`1 + 1`

Executing

  • At build time
  • Live in a browser
- Executable content in - Cells in a Jupyter Notebook (`.ipynb`) - `code-cell` blocks in Markdown files - Inline code using the `eval` role - Can execute at build time with a local Jupyter server and kernel - Can execute on the page by connecting to a Jupyter server and kernel (_e.g._ MyBinder) - Uses - Interactive demos for readers - Curating data stories - Presenting research in a reproducible format

More features!

  • Plugins
  • Cross-book references and embedding
  • Multiple export formats (HTML, pdf, docx, JATS)
- Powerful and flexible plugin system - Add custom functionality and helpers - Can be written in Python or JS - Reusable: content can be referenced or embedded from other Jupyter Books - Contribute to a network of books - Don't reinvent the wheel - Exports to HTML, PDF (LaTeX and Typst), word, JATS - Through intermediate AST - Suitable for websites, portable/printable documents, journals
